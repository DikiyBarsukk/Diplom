Вход в систему

Локальная offline-first версия для дипломной демонстрации. Сервер, агент и аудит работают без внешних CDN.

Стандартный вход для дипломной версии: admin / admin123

Если вход не подходит, значит используется старая SQLite-база с прежней учётной записью.
