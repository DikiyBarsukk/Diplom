BARSUKSIEM

Платформа удалённого аудита и расследования событий ИБ

Offline-first SOC

Оперативный контроль, инциденты и аудит в одном окне.

Стендовая дипломная версия не зависит от внешних CDN, шрифтов и иконок. Всё оформление и логика загружаются локально с сервера.

Панель SOC Главный экран показывает текущий риск, критичные события и свежесть телеметрии.
Быстрый переход к расследованию Из списка инцидентов можно сразу открыть связанные логи и карточку детекта.
Полностью локальный запуск Интерфейс одинаково работает на Ubuntu ВМ и при дополнительном доступе с Windows-хоста по IP сервера.
Авторизация

Вход в BARSUKSIEM

Введите учётные данные администратора или аудитора для доступа к журналам, инцидентам, аналитике и удалённому аудиту.

После входа откроется главная панель SOC.
Стандартный вход для дипломной версии admin / admin123

Если стандартный вход не подходит, значит на сервере уже используется старая SQLite-база с другой учётной записью администратора.
