# Legacy Frontend Archive

Этот каталог хранит исторические frontend-файлы, которые сохранены только как архив.

Правила:

- файлы в этом каталоге не являются частью рабочего UI;
- активные страницы из `web/` не должны их подключать;
- `deploy/server-pack/web` не должен включать этот каталог;
- новые доработки выполняются только в каноническом active runtime-слое.
