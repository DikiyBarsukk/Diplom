# BARSUKSIEM

BARSUKSIEM — дипломная система удалённого аудита журналов событий и визуального анализа инцидентов информационной безопасности.

Проект включает:
- сервер приёма, нормализации и хранения событий;
- агент сбора логов;
- веб-интерфейс SOC с локальным offline-first UI;
- desktop-клиент аудитора на PySide6.

## Быстрый старт для стенда

Для простой дипломной установки используйте готовые Ubuntu-пакеты:
- `deploy/server-pack`
- `deploy/agent-pack`
- `deploy/auditor-pack`

Подробная инструкция по развертыванию на 3 Ubuntu ВМ находится здесь:
- [ИНСТРУКЦИЯ_3_VM_UBUNTU.md](/E:/МТУСИ%202025-2026/Диплом/ПО/ИНСТРУКЦИЯ_3_VM_UBUNTU.md)

## Канонический сценарий доступа

Основной сценарий:
- сервер работает на Ubuntu ВМ;
- веб открывается на Ubuntu-машине по адресу `http://192.168.0.159:8080/login`;
- агент отправляет события на тот же адрес;
- аудитор использует web UI и desktop-клиент.

Дополнительный сценарий:
- тот же web UI можно открыть и с Windows-хоста по IP серверной ВМ `http://192.168.0.159:8080/login`.

Для дипломной версии зафиксированы стандартные учётные данные:

- `GET /api/incidents`

- `POST /api/auth/logout`
