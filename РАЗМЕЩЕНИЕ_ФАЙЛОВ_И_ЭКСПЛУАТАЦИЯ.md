# РАЗМЕЩЕНИЕ ФАЙЛОВ И ЭКСПЛУАТАЦИЯ

Для дипломной версии не нужно вручную разбирать проект по отдельным файлам.

Используйте готовые пакеты из каталога `deploy/`:
- `deploy/server-pack`
- `deploy/agent-pack`
- `deploy/auditor-pack`

## Что куда копировать

### Серверная Ubuntu ВМ
Копировать:
- `deploy/server-pack`

Рекомендуемый путь:
- `/opt/barsuksiem/server-pack`

### Агентная Ubuntu ВМ
Копировать:
- `deploy/agent-pack`

Рекомендуемый путь:
- `/opt/barsuksiem/agent-pack`

### Ubuntu ВМ аудитора
Копировать:
- `deploy/auditor-pack`

Рекомендуемый путь:
- `/opt/barsuksiem/auditor-pack`

## Что уже входит в пакеты

`server-pack` уже содержит:
- `main.py`
- `requirements-server.txt`
- `server/`
- `common/`
- `web/`
- `install.sh`
- `run-server.sh`
- `check-server.sh`
- `data/`

`agent-pack` уже содержит:
- `main.py`
- `requirements-client.txt`
- `client/`
- `common/`
- `install.sh`
- `run-agent.sh`
- `check-agent.sh`

`auditor-pack` уже содержит:
- `main.py`
- `requirements-client.txt`
- `client/`
- `common/`
- `install.sh`
- `run-auditor-gui.sh`
- `check-auditor.sh`

## Что не нужно копировать отдельно

Не нужно вручную переносить:
- `tests/`
- `.venv/`
- `.idea/`
- лишние `.md`-файлы
- временные файлы и кэши

## Как запускать

На каждой Ubuntu ВМ используется один и тот же шаблон:

```bash
chmod +x *.sh
./install.sh
./run-*.sh
```
