# ИНСТРУКЦИЯ

Основной документ для установки дипломной версии BARSUKSIEM теперь один:
- [ИНСТРУКЦИЯ_3_VM_UBUNTU.md](/E:/МТУСИ%202025-2026/Диплом/ПО/ИНСТРУКЦИЯ_3_VM_UBUNTU.md)

Используйте именно его, если вы хотите развернуть систему на трех Ubuntu ВМ:
- сервер
- агент
- аудитор

Короткая логика установки:
1. Скопировать `deploy/server-pack` на серверную ВМ.
2. Скопировать `deploy/agent-pack` на агентную ВМ.
3. Скопировать `deploy/auditor-pack` на ВМ аудитора.
4. На каждой машине выполнить:
   - `chmod +x *.sh`
   - `./install.sh`
   - `./run-*.sh`

Стандартный вход для дипломной версии:
- логин: `admin`
- пароль: `admin123`
