# Инструкция по развертыванию BARSUKSIEM

Основная инструкция по развертыванию проекта теперь находится в [ИНСТРУКЦИЯ_3_VM_LINUX.md](/E:/МТУСИ%202025-2026/Диплом/ПО/ИНСТРУКЦИЯ_3_VM_LINUX.md).

Используйте именно ее, если вы хотите:

- развернуть систему на 3 отдельных виртуальных машинах;
- понять, какие файлы куда копировать;
- пошагово поднять сервер, агента и рабочее место аудитора;
- проверить, что система действительно работает.

Коротко:

- ВМ 1: сервер BARSUKSIEM и web UI
- ВМ 2: агент сбора логов
- ВМ 3: аудитор с браузером и desktop-клиентом
